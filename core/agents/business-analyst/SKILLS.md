# Business Analyst — Available Skills

Skills are invoked on demand. Each skill file is self-contained and can be composed into larger workflows. Every workflow follows three phases: **Pre-work** (investigate and frame the problem), **Execution** (produce the requirements deliverables), **Post-work** (validate that the work is complete and deliverable).

## Pre-work Skills

| Skill | File | When to use |
|---|---|---|
| interview-me | @core/skills/business-analyst/interview-me.md | Extract what a stakeholder actually wants through one-question-at-a-time structured elicitation |
| idea-refine | @core/skills/business-analyst/idea-refine.md | Turn a vague idea into a scoped proposal through divergent then convergent thinking |
| innovation-discovery | @core/skills/business-analyst/innovation-discovery.md | Explore unspoken needs and opportunities before a ticket exists |
| requirements-gathering | @core/skills/business-analyst/requirements-gathering.md | Extract and document requirements from stakeholders |
| process-analysis | @core/skills/business-analyst/process-analysis.md | Map and analyse an existing business process to find gaps or improvements |

## Execution Skills

| Skill | File | When to use |
|---|---|---|
| user-story | @core/skills/business-analyst/user-story.md | Write well-formed user stories with acceptance criteria |
| prd-writing | @core/skills/business-analyst/prd-writing.md | Produce a product requirements document for a feature or initiative |
| roadmap | @core/skills/business-analyst/roadmap.md | Build or update a prioritised product roadmap |

## Post-work Skills

| Skill | File | When to use |
|---|---|---|
| estimation | @core/skills/business-analyst/estimation.md | Estimate effort or complexity to validate that the backlog is realistically deliverable |
| sprint-planning | @core/skills/business-analyst/sprint-planning.md | Organise and sequence requirements into executable sprint increments |

## Composite Workflows

Composite skills chain pre-work, execution, and post-work into a single invocable workflow.

| Skill | File | Phase chain |
|---|---|---|
| requirements-to-prd | @core/skills/business-analyst/requirements-to-prd.md | requirements-gathering (pre-work) → user-story → prd-writing (execution) |
