# Coding Agent Playbook

A carefully curated, non-redundant collection of **SKILLs** and **AGENTs** for AI coding agents (Claude, GitHub Copilot, Codex, etc.), designed to cover all essential practices for professional software development.

## What This Is

**SKILLs** are domain-specific knowledge modules that enable AI agents to handle specialized tasks (e.g., code review, debugging, architecture design, API design) with industry-standard practices.

**AGENTs** are autonomous subagents for multi-step task execution (e.g., codebase exploration, refactoring, testing) that work with or independently of main AI assistants.

Each entry is **complete**, **non-redundant**, and based on real-world software company needs.

## Structure

```
skills/              # Domain-specific expertise modules
agents/              # Autonomous task-execution subagents
examples/            # Real-world usage patterns
LICENSE
README.md
```

## Core Use Cases

- Code review automation
- Debugging & troubleshooting
- Architecture & design decisions
- API & database design
- Refactoring & code quality
- Testing strategies
- Codebase exploration & understanding
- Performance optimization
- Documentation generation
- Security best practices

## Quick Start

1. Browse `skills/` and `agents/` directories for your domain
2. Read the relevant `SKILL.md` or `AGENT.md` documentation
3. Integrate into your AI tool configuration
4. Apply to your workflow

## Principles

- **Concise**: No duplicate content; each skill/agent covers its domain completely without overlap
- **Complete**: Comprehensive coverage of all essential software company practices
- **Practical**: Industry-validated patterns based on real-world development needs

## Contributing

Contributions must follow these rules:

1. No duplication with existing skills or agents
2. Complete, self-contained coverage of the domain
3. Clear documentation with examples
4. Tested with supported AI tools

## Supported AI Tools

- GitHub Copilot (VS Code)
- Claude (via system prompts & custom instructions)
- OpenAI Codex
- Other compatible AI coding assistants
